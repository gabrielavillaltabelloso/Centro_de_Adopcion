#Centro de adopcion

Una solución desarrollada con **Python** y **MySQL** para la gestión de datos, enfocada en la eficiencia y el uso de entornos Linux.

---

## 🚀 Sobre el Proyecto

Este proyecto es una aplicación que conecta una interfaz estilizada con una base de datos relacional. Fue desarrollado enteramente en un entorno **Linux**, aprovechando la potencia de la terminal para la gestión de procesos y el control del flujo de datos.

### 🛠️ Tecnologías utilizadas

* **Lenguaje:** [Python](https://www.python.org/) (Lógica de servidor y conexión).
* **Base de Datos:** [MySQL](https://www.mysql.com/) (Almacenamiento persistente y consultas SQL).
* **Interfaz:** [CSS](https://developer.mozilla.org/es/docs/Web/CSS) (Diseño y maquetación visual).
* **Entorno:** [GNU/Linux](https://www.linux.org/) (Desarrollo y ejecución mediante terminal).
* **Editor:** [VS Code](https://code.visualstudio.com/).

## ✨ Características Principales

* **Persistencia de datos:** Conexión estable con MySQL para guardar información de manera segura.
* **Interfaz limpia:** Diseño visual personalizado mediante CSS.
* **Optimización para Terminal:** Scripts preparados para ser ejecutados y monitoreados desde la CLI de Linux.
* **Consultas Eficientes:** Implementación de CRUD (Crear, Leer, Actualizar y Borrar) directamente en la base de datos.
